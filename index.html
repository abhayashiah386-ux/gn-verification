  1. Go to console.firebase.google.com → sign in with Google
  2. Click "Add project" → name it gn-verification → disable Analytics → Create

  4. Go to Firestore Database → Create database → choose Test mode → Done
  5. Paste each value from firebaseConfig into the fields below → click Connect
